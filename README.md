# serverless-typescript-boilerplate

## Usage

### Setup

```
$ yarn
```

### Local

```
$ yarn run sls offline start
```

### Lint

```
$ yarn tslint
```

### Prettier

```
$ yarn prettier
```

### Test

```
$ yarn test
```

## Deploy

```
$ cp .envrc.sample .envrc
$ direnv allow
```

### Deploy

```
$ yarn sls deploy -v
```
